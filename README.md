⚔️ Nexus RPG - Virtual Tabletop
Nexus RPG é uma plataforma VTT (Virtual Tabletop) moderna e imersiva, focada em trazer a melhor experiência de Dungeons & Dragons 5e para o navegador. Desenvolvido para mestres que exigem controle total e jogadores que buscam imersão visual.

🐲 Funcionalidades Principais
🌌 Imersão Visual & Combate
Dados 3D Realistas: Rolagens de D20 integradas com física via Three.js e Matter.js (estilo Baldur's Gate 3).

Fila de Iniciativa Inteligente: Solicite rolagens de múltiplos alvos simultaneamente. Os jogadores rolam em suas telas e o mestre processa a fila de NPCs de forma fluida.

Sistema de Dano Visceral: Overlay de impacto 2D com animações de sangue e cura, calculando automaticamente a vida dos tokens.

Grid de 1,5m (5ft): Mapa esquadrinhado para medições táticas precisas.

🗺️ Gestão de Mapa (DM Tools)
Fog of War Dinâmico: Revele salas ou esconda perigos usando pincéis, retângulos ou linhas de visão.

Ambientação Global: Controle de iluminação em tempo real (Dia/Noite/Crepúsculo) que afeta a visibilidade de todos.

Régua de Medição: Ferramentas de medição de deslocamento (limite de 9m) e alcance livre.

Áreas de Efeito (AoE): Desenhe círculos, cones e cubos para calcular quem será atingido por aquela Bola de Fogo.

🔊 Sincronização & Áudio
Real-time Sync: Movimentação de tokens, pings e chat sincronizados instantaneamente via WebSockets.

Soundboard Integrado: Músicas de ambiente e efeitos sonoros (SFX) que tocam para todos os jogadores ao mesmo tempo.

Chat Avançado: Comandos de rolagem (/r 1d20+5), sussurros privados (/w) e logs de combate automáticos.

🛠️ Tecnologias Utilizadas
O Nexus foi forjado com as ferramentas mais poderosas do desenvolvimento web moderno:

Frontend: React.js + Vite

Linguagem: TypeScript (Tipagem estrita para segurança de dados RPG)

Estilização: Tailwind CSS

Comunicação: Socket.io

Gráficos 2D/3D: Canvas API, Three.js & Matter.js

Áudio: Howler.js

🚀 Como Executar o Projeto
Pré-requisitos
Node.js (v18 ou superior)

NPM ou Yarn

Instalação
Clone o repositório:

Bash
git clone https://github.com/JoaoVictor/nexus-rpg.git
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
